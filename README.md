# LYCEUM_BACKEND
Lyceum Project Backend

## Clone
### `git clone https://github.com/cwawong/LYCEUM_BACKEND.git`
### `cd LYCEUM_BACKEND`

## Create virtual environment
### `python3 -m pip install --user virtualenv`

## Install packages
### `pip install -r requirements.txt`

## Run
### `python manage.py runserver`
